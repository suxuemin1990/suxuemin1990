## 🧑‍💻 关于我 · About Me

做 **AI 训练平台** 的开发和架构——说白了就是想办法让一堆 GPU 被更多人、更高效地用起来。往下是 GPU 的切分与调度（在啡 [Volcano](https://github.com/volcano-sh/volcano)、[HAMi](https://github.com/Project-HAMi) 的源码），往上是 Kubeflow 那套训练流水线，中间的坑基本都趹过。

更早做 **大数据** 开发，所以对数据量大、链路长的系统不怃。语言不挑，Go / Python / Java / 前端后端，哪个趁手用哪个。

平时爱干两件事：**往下钻**，把框架和内核的原理搞明白；**往外看**，有新东西冒出来就抓过来试试。
