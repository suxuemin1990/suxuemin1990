## 🧑‍💻 关于我 · About Me

👋 你好,我是 **suxuemin1990** —— 一名专注 **云原生基础设施** 与 **AI 平台工程** 的工程师。

日常在 **Kubernetes** 上折腾 **GPU 虚拟化与共享调度**(关注 [Volcano](https://github.com/volcano-sh/volcano)、[HAMi](https://github.com/Project-HAMi) 这类方向),想让 AI 训练与推理在容器里跑得又稳又省资源;同时在研究用 **Kubeflow** 搭建 **MLOps** 流水线,把模型从实验到上线的过程尽量自动化。

我的工作环境是 **WSL + Linux**,信奉「能自动化的绝不手动」——喜欢用脚本和 SSH 把繁琐的事情一次性解决掉。最近在补前端的短板,学习 **React**,顺手把平台的控制台也拿下。
